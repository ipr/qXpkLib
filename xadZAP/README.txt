Summary:
ZAP implementation for qXpkLib to load.

Modifications:
* replace Amiga-specific stuff (lib calling conventions etc.)
* replace hooks with virtual methods&interfaces etc.
* standard typedefs instead of yet another style..
* etc.

Plaholder only currently..
In progress..

Description:
ZAP was lesser-known trackdisk-cruncher (disk image) for transferring disks on Amiga.
It was made as alternative for "Warp", seems to similar to DMS (later?).
